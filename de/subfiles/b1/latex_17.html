1.7 Weitere Pakete

fancyhdr

Mit fancyhdr kann man Kopf- und Fusszeilen variabel definieren. Dafür reicht schon folgende Vorlage mit Beispiel-Einträgen, welche in der Präambel stehen muss.

\pagestyle{fancy} \fancyhf{} \lhead{Schule / Uni} \rhead{Titel} \lfoot{Name} \rfoot{Seite \thepage} \renewcommand{\headrulewidth}{0.4pt} \renewcommand{\footrulewidth}{0.4pt}

xcolor

Wie der Name schon sagt, braucht man xcolor, um die Schriftfarbe zu ändern. Folgende Farben können gebraucht werden: white black red green blue cyan magenta yellow gray darkgray lightgray brown lime olive orange pink purple teal violet. Folgender Code wird dafür gebraucht.

\textcolor{FARBE}{Text}

enumitem

Mit enumitem kann man Aufzählungssymbole variieren und / oder formatieren.

% itemize % Hier können beliebige Symbole eingesetzt werden. \begin{itemize}[label=!] \item Erstes Item \end{itemize} % enumerate % Hier kann man bei beliebiger Zahl beginnen. \begin{enumerate}[start=4] \item Viertes Item \end{enumerate} % Hier können die Zahlen beliebig formatiert werden. \begin{enumerate}[label=\textbf{\arabic*.}] \item Dickes Item \end{enumerate} % Hier kann auch ein Text stehen. \begin{enumerate}[label=\textbf{Aufgabe \arabic*:}] \item Dickes Item \end{enumerate} % Hier sind die Zahlen römisch (klein). \begin{enumerate}[label=(\roman*)] \item Römisches Item \end{enumerate} % Hier sind die Zahlen römisch (gross). \begin{enumerate}[label=\Roman*] \item RÖMISCHES Item \end{enumerate}

hyperref

Mit hyperref können Referenzen zu anderen Kapiteln, Seiten, Bildern etc. erstellt werden.

\section{Abschnitt} \label{sec:abschnitt} % wichtig Eine markierte Gleichung: \begin{equation} E = mc^2 \label{eq:demo} % wichtig \end{equation} \section{Listen mit Hyperlinks} \begin{enumerate}[label=\arabic*.] \item Link zu Abschnitt \ref{sec:abschnitt} \item Link zu Gleichung \ref{eq:demo} \item Externer Link: \href{https://google.ch}{Google} \item Externer Link mit sichtbarer URL: \url{https://www.wikipedia.org} \end{enumerate}

float

Durch float kann man grundsätzlich Gleitumgebungen automatischen platzieren lassen. Diese Änderung beeinflusst nur die erste Zeile der Gleitumgeungen. Hier am Beispiel von Figuren:

\begin{figure}[H] % Das H sorgt nun für die optimale Platzierung der Gleitumgebung.

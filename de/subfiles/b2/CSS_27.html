2.7 Grösse und Overflow

Einführung

Mit CSS lassen sich Elemente in der Grösse steuern. width und height legen feste Dimensionen fest, während min-width/max-width Grenzen definieren. overflow bestimmt, wie Inhalte ausserhalb des Containers angezeigt werden: hidden, auto oder scroll. Text kann mit word-break oder overflow-wrap umgebrochen werden, und text-overflow: ellipsis zeigt "..." für abgeschnittenen Text an.

.box { width: 200px; height: 50px; max-width: 300px; min-width: 100px; overflow: hidden; word-break: break-word; overflow-wrap: break-word; text-overflow: ellipsis; }

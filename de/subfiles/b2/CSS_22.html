2.2 Grundlagen

Einführung

CSS kann auf drei Arten eingebunden werden:
  • Inline: direkt im HTML-Element mit style=""
  • Im <style>-Tag: im <head> der HTML-Datei
  • Externe Datei: .css-Datei über <link> eingebunden
Syntax: Ein CSS-Regelwerk besteht aus einem Selektor und Eigenschaften.

Selektor { property: value; } /* Beispiel: */ p { color: red; font-size: 16px; }

Box Modell

Jedes HTML-Element wird als Rechteck dargestellt, bestehend aus:
  • content: eigentlicher Inhalt (Text, Bild)
  • padding: Abstand zwischen Inhalt und Rahmen
  • border: Rahmenlinie um das Element
  • margin: Abstand zum nächsten Element
  • box-sizing und border-box: sorgt dafür, dass Padding und Border in der Gesamtbreite enthalten sind

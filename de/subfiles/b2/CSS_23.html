2.3 Text und Inhalt

Varianten

Mit CSS kann das Erscheinungsbild von Text und Schriftarten präzise gesteuert werden. font-family legt die Schriftart fest, font-size die Schriftgrösse, font-weight die Stärke (z. B. normal, bold). line-height steuert den Zeilenabstand, text-align die Ausrichtung. Mit text-decoration können Unterstreichungen oder Durchstreichungen hinzugefügt werden. white-space regelt den Umgang mit Leerzeichen und Zeilenumbrüchen.

p { font-family: Arial, sans-serif; /* Schriftart */ font-size: 16px; /* Schriftgrösse */ font-weight: bold; /* Schriftstärke */ line-height: 1.5; /* Zeilenhöhe */ text-align: center; /* Textausrichtung */ text-decoration: underline; /* Unterstreichung */ white-space: nowrap; /* keine Umbrüche */ }

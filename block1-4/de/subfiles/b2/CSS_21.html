2.1 Was ist CSS?

Einführung

CSS (Cascading Style Sheets) ist die Sprache, mit der das Aussehen von HTML-Inhalten gestaltet wird. Sie trennt Layout, Farben, Schriftarten und Formatierungen vom eigentlichen Inhalt.

Warum CSS?

Mit CSS lässt sich das Design einer Website zentral steuern, wodurch Änderungen schnell umgesetzt werden können, ohne den HTML-Code mehrfach anzupassen.

Anwendungsbereiche

  • Gestaltung von Text, Farben, Hintergründen und Rahmen
  • Layouts: Flexbox, Grid, Positionierung
  • Responsive Design für verschiedene Bildschirmgrössen
  • Animationen und Übergänge

Fazit

CSS ist unverzichtbar für moderne Webseiten. Es sorgt für Flexibilität, Einheitlichkeit und professionelle Darstellung von Inhalten.

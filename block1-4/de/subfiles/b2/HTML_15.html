1.5 Links und Navigation

Links

Mit dem HTML-Element <a> können Verknüpfungen zu anderen Seiten oder Ressourcen erstellt werden. Links können intern (innerhalb derselben Website) oder extern (zu einer anderen Website) sein.

Target-Attribute

Mit target="_blank" öffnet sich ein Link in einem neuen Browser-Tab oder Fenster. Standardmässig wird ein Link im selben Fenster geöffnet.

Ankerlinks

Ankerlinks springen zu einem bestimmten Abschnitt innerhalb derselben Seite. Dazu wird die ID eines Elements im href verwendet, z. B. <a href="#section1">.

Beispiel

Im Folgenden ist ein möglicher Anwendungsbeispiel.

<!DOCTYPE html> <html lang="de"> <head> <meta charset="UTF-8"> <title>Links & Navigation</title> </head> <body> <h2>Externer Link</h2> <a href="https://rr-utilities.github.io/forumdigitalis/" target="_blank">Beispielseite öffnen</a> <h2>Interner Link</h2> <a href="about.html">Zur About-Seite</a> <h2>Ankerlink</h2> <a href="#section1">Springe zu Abschnitt 1</a> <h2 id="section1">Abschnitt 1</h2> <p>Inhalt von Abschnitt 1.</p> </body> </html>

2.5 Positionierung

Einführung

Mit CSS kann die Position von Elementen gesteuert werden. position legt die Art der Positionierung fest: static (Standard), relative (relativ zur eigenen Position), absolute (relativ zum nächsten positionierten Vorfahren), fixed (fixiert im Viewport) und sticky (klebt beim Scrollen innerhalb des Containers). Mit top, left, right und bottom kann der Abstand gesetzt werden.

div { position: absolute; /* absolute Position */ top: 50px; /* Abstand von oben */ left: 100px; /* Abstand von links */ }

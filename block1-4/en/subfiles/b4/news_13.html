1.3 HTML, CSS etc.: Block 2

Block 2 is now online.

This block contains a guide to HTML, CSS and TikZ (addition).
HTML HTML (HyperText Markup Language) is the basic language of the web. It describes the structure and content of a web page, e.g. headings, paragraphs, images, links and forms.
CSS CSS (Cascading Style Sheets) is the design language of the web. It determines how content defined with HTML looks, e.g. colours, fonts, spacing, layouts and responsive design.
TikZ (Addition) TikZ is a LaTeX package for creating graphics. It is used to precisely define diagrams, vector graphics, mathematical drawings and illustrations directly in LaTeX documents.

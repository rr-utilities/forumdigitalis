1.9 Formulare

Einführung

HTML-Formulare werden mit dem <form>-Element erstellt. Innerhalb eines Formulars können verschiedene Eingabefelder verwendet werden, z. B. text, email, password, date oder number. Mit <label> werden die Eingabefelder beschriftet, und <button> dient zum Abschicken des Formulars. Attribute wie required erzwingen eine Eingabe, während placeholder einen Platzhaltertext anzeigt.

Beispiel

Das folgende Beispiel zeigt ein einfaches Formular mit verschiedenen Eingaben.

<!DOCTYPE html> <html lang="de"> <head> <meta charset="UTF-8"> <title>Formulare</title> </head> <body> <form action="#" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name" placeholder="Ihr Name" required> <br> <label for="email">E-Mail:</label> <input type="email" id="email" name="email" placeholder="email@beispiel.com" required> <br> <label for="password">Passwort:</label> <input type="password" id="password" name="password" required> <br> <label for="geburtstag">Geburtsdatum:</label> <input type="date" id="geburtstag" name="geburtstag"> <br> <label for="anzahl">Anzahl:</label> <input type="number" id="anzahl" name="anzahl" min="1" max="10"> <br> <button type="submit">Absenden</button> </form> </body> </html>

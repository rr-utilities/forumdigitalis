2.1 What is CSS?

Introduction

CSS (Cascading Style Sheets) is the language used to style HTML content. It separates layout, colors, fonts, and formatting from the actual content.

Why CSS?

CSS allows centralized control of a website’s design, enabling quick updates without changing the HTML multiple times.

Areas of application

  • Styling text, colors, backgrounds, and borders
  • Layouts: Flexbox, Grid, positioning
  • Responsive design for different screen sizes
  • Animations and transitions

Conclusion

CSS is essential for modern websites. It provides flexibility, consistency, and professional presentation of content.

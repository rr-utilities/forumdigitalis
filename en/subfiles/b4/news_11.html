1.1 Launch of Forum digitalis

The Forum digitalis officially opens its digital doors.

Digitalisation is progressing and so are digital things like coding and its products, such as HTML websites (with CSS and JavaScript) or LaTeX documents, which is not directly "coding", but why not? So everyone has free access to coding and co.
This forum should help people who either want to start coding or already know how to code but need to look something up. It is therefore an intuitive step-by-step learning module and a reference work at the same time. All the content will be available on this website for free and without advertising, but not all of it will be published at once.

Block 1 will be released in January 2026 and the others will be announced soon.

1.2 Latex, BibTeX etc.: Block 1

Block 1 is now online.

This block contains a guide to LaTeX, BibTeX, and Markdown.
LaTeX A typesetting system for scientific papers. Focus on perfect typography, formulas, and references. You write code, which is then compiled into a professional PDF.
BibTeX An add-on to LaTeX for literature management. You save sources in a .bib file and LaTeX automatically generates a bibliography and citations from it.
Markdown A very simple markup language for formatted text (headings, bold, lists). Often used for notes, GitHub files, web pages, and documentation. No compilation required, directly readable.
